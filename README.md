# **Diploma-project**
____
## Taxi Booking System
____
## Tech stack:
1. Django, Django-restframework
2. PostgreSQL
____
### Launch of the project:
''' 
pip install requirements.txt
'''
____
### Functional:
1. Регистрация/Авторизация водителя.
2. Регистрация/Авторизация пассажира.
3. Возможность отправлять запрос на бронь машины.
4. Оценка поездки.
